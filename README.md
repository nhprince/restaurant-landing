# MAISON — Restaurant Landing Page

A premium, fully responsive restaurant landing page built with Next.js, Framer Motion, and Tailwind CSS.

## Design

- **Theme**: Dark, minimal, premium
- **Colors**: Gold accent (#c9a96e) on deep black
- **Typography**: Playfair Display (serif) + Inter (sans)
- **Animations**: Framer Motion scroll-triggered animations
- **Sections**: Hero, About, Menu, Gallery, Testimonials, Reservation, Contact, Footer

## Quick Start

```bash
cd frontend
npm install
npm run dev
```

## Deploy

```bash
npm run build
# Output: out/ (static export)
```

## Tech Stack

- Next.js 14 (static export)
- Tailwind CSS
- Framer Motion
- Lucide React icons
